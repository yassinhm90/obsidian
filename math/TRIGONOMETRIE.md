4) LES RELATION TRIGONOMETRIQUE 
- relation entre -x et x 

	cos -x = cos x
	sin -x = -sin x
	tan -x = -tanx


- relation entre $\pi$/2 -x et x


	cos ( $\pi/2 - x$ ) = sin x
	sin ( $\pi/2 - x$  ) = cos x 
	tan ( $\pi/2 - x$  ) = 1/tan x


- relation entre  $\pi/2$  + x  et  x

	cos ( $\pi/2 + x$ ) = -sin x 
	sin ( $\pi/2 + x$ ) = cos x 
	tan ( $\pi/2 + x$ ) = -1/tan x

  
- relation entre  $\pi$ - x et x

	cos ( $\pi - x$ ) = -cos x 
	sin ( $\pi - x$ ) = sin x 
	tan ( $\pi - x$ ) = -tan x


- relation entre $\pi$ + x et x

	cos ( $\pi + x$ ) = -cos x
	sin ( $\pi + x$ ) = -sin x
	tan ( $\pi + x$ ) = tan x
